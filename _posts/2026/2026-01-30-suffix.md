我们知道一个单词是由**前缀+词根+后缀**组成。

1. 前缀控制着单词的情感和方向
2. 词根是单词的核心和灵魂
3. 后缀控制着单词的词性

下面我们讲述一下英语中一些常见的后缀

### 英语后缀的主要来源
#### (1) **古英语 / 日耳曼来源**
英语最早是古英语（Old English，约5–11世纪）的发展，它本身是日耳曼语系的一支。很多基础性的后缀来自古英语，例如：

+ **-ness** → 名词后缀，表示状态或性质
    - happy → happiness
    - dark → darkness
+ **-ly** → 副词后缀
    - quick → quickly
+ **-dom** → 名词后缀，表示领域、状态
    - king → kingdom

#### (2) **拉丁语和法语来源**
1066年诺曼征服英格兰后，英语大量吸收了**法语**（本身有很多拉丁语成分）词汇，这给英语带来了大量学术、法律、宗教、抽象概念的后缀：

+ **-tion / -sion** → 名词后缀
    - act → action
    - expand → expansion
+ **-able / -ible** → 形容词后缀
    - read → readable
    - access → accessible
+ **-al** → 形容词/名词后缀
    - tradition → traditional
    - region → regional

#### (3) **希腊语来源**
英语在科学、医学、哲学等领域借用了大量希腊语词根和后缀：

+ **-ology** → 学科或学问
    - biology, psychology
+ **-phobia** → 恐惧症
    - arachnophobia, xenophobia
+ **-graphy** → 写作或描绘
    - geography, biography

## 英语常用后缀
- **-ness**
  - 来源：古英语（日耳曼）
  - 功能：名词，表示状态、性质
  - 例词：happiness（幸福），darkness（黑暗），kindness（善良）

- **-ly**
  - 来源：古英语（日耳曼）
  - 功能：副词，表示方式、程度
  - 例词：quickly（快速地），slowly（缓慢地），happily（快乐地）

- **-dom**
  - 来源：古英语（日耳曼）
  - 功能：名词，表示领域、地位、状态
  - 例词：kingdom（王国），freedom（自由），wisdom（智慧）

- **-ship**
  - 来源：古英语 / 日耳曼
  - 功能：名词，表示身份、技能、关系
  - 例词：friendship（友谊），leadership（领导地位/才能），partnership（合伙关系）

- **-tion / -sion**
  - 来源：法语 / 拉丁语
  - 功能：名词，表示动作或结果
  - 例词：action（行动），expansion（扩大），tension（紧张）

- **-ment**
  - 来源：法语 / 拉丁语
  - 功能：名词，表示动作或状态
  - 例词：development（发展），agreement（协议），treatment（对待）

- **-able / -ible**
  - 来源：法语 / 拉丁语
  - 功能：形容词，可……的、能……的
  - 例词：readable（可读的），accessible（可到达的），flexible（灵活的）

- **-al**
  - 来源：法语 / 拉丁语
  - 功能：形容词 / 名词，与……相关
  - 例词：regional（地区的），traditional（传统的），arrival（到达）

- **-ery**
  - 来源：古法语 / 拉丁语
  - 功能：名词，地方、行为、职业
  - 例词：bakery（面包店），robbery（抢劫），pottery（陶器）

- **-ary**
  - 来源：拉丁语 / 法语
  - 功能：名词 / 形容词，与……有关、场所、集合体
  - 例词：library（图书馆），missionary（传教士），budgetary（预算的）

- **-ent**
  - 来源：拉丁语（经法语）
  - 功能：形容词……的 / 名词（人或物）
  - 例词：dependent（依赖的），efficient（有效的），student（学生）

- **-ant**
  - 来源：拉丁语（经法语）
  - 功能：形容词……的 / 名词（人或物）
  - 例词：resistant（有抵抗力的），important（重要的），assistant（助手）

- **-ize / -ise**
  - 来源：法语 / 拉丁语
  - 功能：动词，使……化
  - 例词：modernize（使现代化），realize（意识到），organize（组织）

- **-en**
  - 来源：古英语 / 日耳曼
  - 功能：动词，使……
  - 例词：strengthen（加强），darken（变暗），lengthen（延长）

- **-ic**
  - 来源：法语 / 拉丁语
  - 功能：形容词，……的
  - 例词：poetic（诗的），scientific（科学的），historic（历史上重要的）

- **-ology**
  - 来源：希腊语
  - 功能：名词，学科、学问
  - 例词：biology（生物学），psychology（心理学），geology（地质学）

- **-phobia**
  - 来源：希腊语
  - 功能：名词，恐惧症
  - 例词：arachnophobia（蜘蛛恐惧症），xenophobia（仇外）

- **-graphy**
  - 来源：希腊语
  - 功能：名词，写作或描绘
  - 例词：geography（地理），biography（传记），cartography（制图）

- **-ist**
  - 来源：法语 / 希腊语
  - 功能：名词，从事某职业或学科的人
  - 例词：artist（艺术家），scientist（科学家），biologist（生物学家）

- **-ward / -wards**
  - 来源：古英语
  - 功能：副词 / 形容词，方向
  - 例词：forward（向前），backwards（向后），homeward（向家）

- **-ful**
  - 来源：古英语
  - 功能：形容词，充满……的
  - 例词：joyful（高兴的），careful（小心的），hopeful（充满希望的）

- **-less**
  - 来源：古英语
  - 功能：形容词，没有……的
  - 例词：powerless（无力的），hopeless（无望的），careless（粗心的）

- **-ish**
  - 来源：古英语
  - 功能：形容词 / 动词，带有……性质、稍微……
  - 例词：childish（孩子般的），bluish（带蓝色的）

- **-ity**
  - 来源：拉丁语（经法语）
  - 功能：名词，状态、性质、抽象概念
  - 例词：reality（现实），ability（能力），security（安全），stability（稳定）
